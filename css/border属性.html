    # 为什么border属性的默认宽度大小是medium3px？

    因为：border-style:double想要完全展示必须宽度为3px,因为其双边框表现规则为两边宽度永相等，中间宽度加减一

    三道杠小技巧

    height:10px;
    width: 50px;
    border-top: 10px double;
    border-bottom: 10px solid;
}
